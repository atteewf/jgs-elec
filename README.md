## 🛠️ Lancer en local

**Prérequis :** Node.js 18+, pnpm

```bash
git clone https://github.com/atteewf/jgs-elec.git
cd jgs-elec
pnpm install
pnpm dev
```

Puis ouvrir : `http://localhost:3000`
